# Welcome to Aran API Sentinel

**Aran API Sentinel** is your comprehensive multi-tenant API security and governance platform, designed to protect and manage APIs in modern, distributed environments.

![Aran API Sentinel Dashboard](/screenshots/dashboard.png)
*Real-time API security monitoring and governance*

## 🚀 **What is Aran API Sentinel?**

Aran API Sentinel is a next-generation API security platform that combines:

- **🔐 Multi-Tenant Security**: Complete tenant isolation with PocketBase backend
- **🤖 MCP Security**: Comprehensive Model Context Protocol protection for AI/ML applications
- **📊 Real-time Monitoring**: Live API health and security analytics
- **🛡️ Threat Detection**: Rule-based analysis and anomaly detection
- **📚 API Governance**: Centralized API catalog and documentation management

## 🎯 **Key Features**

### **🔍 API Discovery & Catalog**
![API Discovery](/screenshots/api-discovery.png)
*Automated API discovery using pattern matching and rule-based analysis*

Discover and catalog all your APIs automatically:
- **Pattern-based Discovery**: Find APIs in traffic data and code
- **Centralized Catalog**: Organize APIs with metadata and documentation
- **Search & Filter**: Find APIs quickly with advanced filtering
- **Version Management**: Track API versions and changes

### **🛡️ Threat Detection & Security**
![Threat Detection](/screenshots/threat-detection.png)
*Real-time threat detection with comprehensive security analysis*

Protect your APIs from threats:
- **Real-time Monitoring**: Live threat detection and alerting
- **Anomaly Detection**: Identify unusual API patterns
- **Security Scoring**: Risk assessment for API endpoints
- **Incident Response**: Automated threat mitigation

### **🤖 MCP Security for AI/ML**
![MCP Security](/screenshots/mcp-security.png)
*Comprehensive Model Context Protocol security monitoring*

Secure AI applications using Model Context Protocols:
- **Tool Poisoning Detection**: Identify compromised MCP tools
- **Line Jumping Prevention**: Stop AI bypassing validation
- **Tool Shadowing Detection**: Detect malicious tool impersonation
- **Prompt Injection Protection**: Prevent malicious prompt injection
- **Authorization Enforcement**: Ensure proper AI permissions

### **📊 Dashboard & Analytics**
![Dashboard](/screenshots/dashboard.png)
*Comprehensive dashboard with real-time metrics and insights*

Monitor your API ecosystem:
- **Real-time Metrics**: Live API health and performance data
- **Security Posture**: Overall security status and trends
- **Usage Analytics**: API usage patterns and optimization
- **Custom Dashboards**: User-defined monitoring views

### **🔐 Multi-Tenant Architecture**
![Login](/screenshots/login.png)
*Secure multi-tenant authentication with company-specific access*

Built for enterprise scale:
- **Tenant Isolation**: Complete data separation between companies
- **Role-Based Access**: Granular permissions per user
- **Company Branding**: Custom logos and settings per tenant
- **Audit Logging**: Comprehensive activity tracking

## 🚀 **Quick Start**

### **1. Installation**
```bash
# Clone the repository
git clone https://github.com/aran-api-sentinel/aran.git
cd aran

# Install dependencies
npm install
```

### **2. Start the Application**
```bash
# Start both PocketBase and Next.js
npm run dev:full
```

### **3. Access the Platform**
- **Application**: http://localhost:9002
- **PocketBase Admin**: http://127.0.0.1:8090/_/

### **4. First Setup**
1. Create your first company in PocketBase admin
2. Add users with appropriate roles
3. Start discovering and securing your APIs
